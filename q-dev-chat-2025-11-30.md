## Passos para fazer funcionar:

### 1. **Instalar dependências do backend**
```bash
cd backend
npm install
```

### 2. **Configurar o banco PostgreSQL**
- Instale o PostgreSQL
- Crie um banco chamado `livraria`
- Execute este SQL para criar a tabela:
```sql
CREATE TABLE livros (
    id SERIAL PRIMARY KEY,
    nome VARCHAR(255) NOT NULL,
    preco DECIMAL(10,2) NOT NULL,
    descricao TEXT,
    estoque INTEGER NOT NULL
);
```

### 3. **Iniciar o backend**
```bash
cd backend
node server.js
```

### 4. **Abrir o frontend**
- Abra o arquivo `index.html` no navegador
- Ou use um servidor local como Live Server

### 5. **Configurar credenciais do banco**
Se suas credenciais do PostgreSQL forem diferentes, edite o arquivo `backend/routes/livros.js`:
```javascript
const pool = new Pool({
  user: 'seu_usuario',
  host: 'localhost',
  database: 'livraria',
  password: 'sua_senha',
  port: 5432,
});
```

Agora seu site conseguirá:
- Carregar livros do banco de dados
- Cadastrar novos livros
- Editar livros existentes
- Excluir livros
- Todas as operações serão persistidas no PostgreSQL
